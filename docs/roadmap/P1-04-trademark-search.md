# P1-04: Trademark Search

Engage trademark attorney to search all marks before public use. A single engagement covers all seven.

## Marks to Search
1. "Tomato Warning" (brand name)
2. "Coastal Calm" (EF-1)
3. "Gulf Breeze" (EF-2)
4. "Squall Line" (EF-3)
5. "Supercell" (EF-4)
6. "Ground Zero" (EF-5)
7. "Chase the storm." (tagline)

## Risk Flags
- "Ground Zero" — common term with prior use in multiple contexts
- "Supercell" — common weather/gaming term
- These may require backup names

## Deliverables
- Trademark clearance report for all 7 marks
- Recommendation on registrability
- Filing applications for cleared marks
