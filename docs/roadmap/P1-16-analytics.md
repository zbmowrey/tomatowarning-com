# P1-16: Configure Site Analytics & Conversion Tracking

Set up analytics to measure the three KPIs that define Phase 1 success.

## KPIs to Track
1. Consumer "Notify Me" signups (500+ target, Gulf Coast FL zip codes priority)
2. Retailer interest signups (50+ target)
3. Nonprofit/org interest signups (10+ target)

## Deliverables
- Analytics platform configured (GA4 or privacy-respecting alternative like Plausible)
- Conversion events for all three signup types
- UTM strategy for traffic source attribution
- Basic dashboard for KPI monitoring

## Notes
- Gap item: without tracking, cannot optimize CTAs or prove demand to retail partners
- Should be configured during site build, go live with the site
