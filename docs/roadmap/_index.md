# Tomato Warning — Product Roadmap

## Overview

44 items across 3 phases, transforming Tomato Warning from a domain name into a live branding site and eventually a shipped product.

**The branding site has one job:** Build three lists that prove demand.
- 500+ consumer "Notify Me" signups (Gulf Coast FL priority)
- 50+ retailer interest signups
- 10+ nonprofit/org fundraising partner signups

## Phase 1: Branding Site (22 items)

Build the branding website: hero, product line (coming soon), retailer CTA, nonprofit CTA, mission page, press page, two mailing lists.
