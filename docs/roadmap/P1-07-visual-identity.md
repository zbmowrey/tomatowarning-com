# P1-07: Establish Visual Identity System

Design the complete visual identity system. This is on the critical path — all subsequent design work depends on it.

## Deliverables

### Color Palette
- Storm charcoal base (#2C2C2C)
- Premium white (#F5F0EB)
- Radar teal accent (#00B4D8)
- EF gradient: EF-1 amber (#D4A017) → EF-2 deep orange (#CC5500) → EF-3 rust red (#8B2500) → EF-4 dark crimson (#5C0A0A) → EF-5 near-black (#1C0A0A)

### Typography
- Bold condensed sans or slab serif for display (weather-alert authority)
- Clean grotesque for body
- No script fonts (deliberate departure from farm-stand artisan aesthetic)

### Storm Imagery System
- Doppler radar texture patterns
- Isobar/pressure line design elements
- Vortex mark direction (feeds into P1-11 logo design)

### Anti-Southwestern Rules
1. No cactus imagery
2. No desert earth-tone palette
3. No handwritten/script fonts
4. No pepper illustration as primary element
5. No state flag imagery

## Notes
- Reference categories: craft spirits, premium hot sauce, not traditional salsa
- The EF-5 near-black must read as sophisticated, not threatening
- This system governs label design, website design, all marketing materials
