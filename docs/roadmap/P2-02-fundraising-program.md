# P2-02: Build Storm Chasers Fund Fundraising Program

Full operational program: consignment model, custom label capability, campaign materials.

## Program Structure
- 50% margin to organizations
- No upfront cost — consignment with 30-day return window
- Custom label at 48-jar minimum
- 3-4 week campaign cycles
- Marketing kit: flyer templates, social post copy, email templates, sales tracker

## Deliverables
- Program operations manual
- Custom label printing workflow
- Consignment logistics and return process
- Marketing kit (digital + print-ready)
- Program coordinator onboarding process
- Impact summary template ("Your campaign sold X jars, raised $Y...")
