# P1-15: Produce Product Mockup Renders

Create high-resolution rendered jar mockups for all five EF levels. This is a shared dependency — the hero, product pages, retailer PDF, and press page all need these renders.

## Deliverables
- Rendered 16oz glass jar mockups for EF-1 through EF-5
- Multiple orientations per variety: front-facing, angled, lifestyle/context
- Label design applied using visual identity (P1-07) and logo (P1-11)
- High-res versions for press use (P1-22)
- Web-optimized versions for site use

## Specifications
- 16oz glass jar, 82mm wide-mouth, straight-sided, clear glass
- Label must show: EF level, product name, heat descriptor, Scoville range, brand logo, charitable mission callout
- EF color coding: each variety's label uses its EF gradient color
- Storm imagery elements from visual identity system

## Notes
- Blocked by P1-07 (visual identity — colors and imagery) and P1-11 (logo)
- Blocks P1-19 (retailer CTA — PDF needs renders), P1-20 (product pages), P1-21 (hero), P1-22 (press page)
- This is on the critical path — prioritize render quality and timeline
- 3D rendering or styled photography with mockup labels are both viable approaches
