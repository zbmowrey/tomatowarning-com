# P2-09: Design QR Code Story Layer per Jar

Per-variety content pages linked from jar labels via QR code. Mobile-first design, updated seasonally.

## Per-Variety Page Content
- Origin of the variety name
- Key ingredients with sourcing notes
- Suggested uses/pairings
- Giving impact update ("This batch helped fund $X of disaster recovery")
- Recipe

## Notes
- Design URL architecture during Phase 1
- Build content pages when product is in production
- Batch-linked giving impact requires charitable program to be operational
